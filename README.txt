En este repositorio de github están:
 
- El cliente, que seria la carpeta client que es la parte frontend de la web hecha con angular.
- El backend que es la parte hecha con node.js.
- La base de datos de mongoDB que esta en la carpeta DB/
- Y, por último, la memoria y el video aparte de la explicación

-Para arrancar el proyecto tendríamos que tener instalado mongoDB y copiar los datos a C:/data/db y luego arrancar el mongod.exe, una
vez hecho esto tendríamos que arrancar el backend con npm start y por último el cliente con ng serve -o
